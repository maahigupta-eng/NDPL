Nirmal Command Center

How to deploy:
1. Upload this whole folder to Netlify, Vercel, GitHub Pages, or any static hosting service.
2. Open index.html locally to test.
3. Default password: NDPL2026
4. To change password, open index.html and search for APP_PASSWORD.

Important:
This is simple client-side password protection. It is suitable for a private family/demo link, not for highly confidential company data.
The dashboard does not fabricate information. It only calculates from imported CSV / Google Sheet data.

Google Sheets import:
Publish the sheet to web as CSV or use a Google Sheet CSV export link, then paste the URL in Data Import.
