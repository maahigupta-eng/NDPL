NirmalOS / Nirmal Command Center

How to deploy:
1. Upload index.html, assets folder, and data folder to GitHub.
2. In the GitHub repository, go to Settings > Pages.
3. Choose Deploy from branch > main > /(root) > Save.
4. Open the live GitHub Pages link after 1-3 minutes.

Default password: NDPL2026
To change password: open index.html and search for APP_PASSWORD.

Important:
- This is a static website with simple client-side password protection. It is fine for a family/demo link, not for confidential company deployment.
- Live dashboard mode does not fabricate information. It only calculates from imported CSV / Google Sheet data.
- Demo Experience uses clearly labeled fabricated sample data to show how the site looks when running.

Google Sheets import:
Publish the sheet to web as CSV or use a Google Sheet CSV export link, then paste the URL in Data Import.
